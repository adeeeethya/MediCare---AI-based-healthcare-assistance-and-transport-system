MediCare – AI-Based Healthcare Assistance and Transport System
Overview
MediCare is a full-stack healthcare assistance platform designed to improve accessibility for elderly individuals. The system connects users with nearby caretakers and enables quick access to healthcare services through a simple and intuitive interface. It focuses on reducing response time and improving coordination using real-time tracking and intelligent allocation.

Features
Real-time booking of healthcare and transport services

GPS-based tracking of caretakers

Automatic allocation of the nearest available caretaker

Role-based access for users, caretakers, and administrators

Simple interface designed for ease of use

Secure authentication and data handling

Tech Stack
Frontend: React

Backend: FastAPI

Database: PostgreSQL

APIs: RESTful services with real-time data handling

How It Works
Users can request assistance through the platform, and the system identifies the nearest available caretaker using location data. The caretaker is assigned automatically, and the user can track their movement in real time. All interactions are handled through a structured backend and database system to ensure reliability and performance.
